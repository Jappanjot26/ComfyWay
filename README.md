# ComfyWay - Your Ultimate Stay Experience

ComfyWay is a modern vacation rental platform, allowing users to find, book, and manage stays at unique accommodations worldwide. Whether you're a traveler looking for a cozy getaway or a host wanting to rent out your space, ComfyWay makes the process seamless and secure.

## Features (Under Development)

### For Guests:
- Browse and book accommodations worldwide
- View high-quality images and detailed property descriptions
- Secure online payments and booking confirmation
- Leave reviews and ratings after your stay
- Advanced search filters (location, price, amenities, etc.)

### For Hosts:
- List properties with images, descriptions, and pricing
- Manage availability with an interactive calendar
- Set pricing, discounts, and cancellation policies
- Secure verification and identity checks
- Track earnings and booking statistics

### Other Key Features:
- Secure user authentication (Sign up, Login, OAuth)
- Interactive maps for location-based property search
- Messaging system for guest-host communication
- Multi-language and currency support

## Tech Stack

- **Frontend:** React, Tailwind CSS
- **Backend:** Node.js, Express, MongoDB

## Getting Started

### 1. Clone the Repository
```sh
git clone https://github.com/Jappanjot26/ComfyWay.git
cd ComfyWay
```

### 2. Install Dependencies
```sh
npm install  # or yarn install
```

### 3. Set Up Environment Variables
Create a `.env` file and add your API keys:
```sh
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
```

### 4. Run the Application
```sh
npm run dev
```

### 5. Run the server
```sh
npm run server
```
